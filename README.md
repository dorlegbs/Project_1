# BrandWatch
Produzido por Gabriela Dorle - Estudante da Fundação Getúlio Vargas

O App tem a função de analisar o aumento no número de menções de uma determinada marca nas redes a partir da NewsApi. Ela permite o acesso, em tempo real, à milhares de dados dispostos em diferentes fontes na internet e, a partir disso, o programa calcula a diferença entre as menções antigas e as menções mais recentes.
Somado a isso, a API do Telegram, também usada, permite que o programa envie um alerta para o usuário quando houver um aumento considerável no número de menções. Esse aviso, então, garante que o usuário esteja ciente, ao vivo, dos pronunciamentos sobre a marca de interesse, podendo identificar uma crise a partir do aumento sinalizado.
Portanto, o programa permite uma rápida leitura do cenário em que a marca está inserida, possibilitando uma resposta mais rápida e eficaz diante de uma movimentação incomum no padrão de menções.
